# onefacepass
为[onefacepass](https://github.com/onefacepass)项目写的UI

## 开发环境

opencv 346
Qt 5.13.0 (msvc 2017 64bit)

## 工程导入

在代码根目录下创建 sample 目录，其中内容详见 https://github.com/onefacepass/face_recognition
QtCreator导入OneFacePass.pro文件
